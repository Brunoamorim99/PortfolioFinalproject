# Portfolio Website

This is the repository for my portfolio website built with Next.js, TypeScript, and Tailwind CSS. The design is based on Figma mockups.

## Project Structure

- `app/` - Next.js app directory with components
- `components/` - Reusable UI components
- `public/` - Static assets
- `styles/` - Tailwind CSS custom styles
- `package.json` - Project dependencies and scripts

## Getting Started

1. Install dependencies with `npm install`
2. Run development server with `npm run dev`
3. Build for production with `npm run build`

## Technologies Used

- Next.js 14
- React 18
- TypeScript 5
- Tailwind CSS 4
- Figma for design
